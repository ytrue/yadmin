# 1.简介
yadmin 基于springboot 开发后台

[后台基于 Ant Design Vue  hi实现 ](https://github.com/ytrue/yadmin-web)

# 2. 目录结构

```
yadmin-admin			  	后台
yadmin-api					api
yadmin-monitor		   		Spring Boot Admin
yadmin-common		 	    公共
yadmin-sys			   	    系统  
yadmin-quartz		 	    定时任务
yadmin-security	   		    oauth2安全	
yadmin-tools			   	工具库
```
